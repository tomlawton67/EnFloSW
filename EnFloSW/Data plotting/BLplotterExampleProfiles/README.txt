This folder contains an example BL target file, "Example.xls".

Edit this file, and save with a new name, to a location of your choice, then point to that folder with the BLplotter subVI: "GatherBLTargets"

This file is also kind-of directly plottable in ResultsFilePlotter. It can have problems, if edited in Excel, with many blank columns, corresponding to the tab characters filling out the space under the asterisked columns…

Files can contain /just/ and n value, or /just/ TI, if you want…
